# **App Name**: CampusConnect

## Core Features:

- Role-Based Authentication: Secure login system with distinct dashboards and privileges for administrators and students.
- Event Listing: Display upcoming and past events with detailed information, including date, time, location, description, and organizers.
- Event Registration: Allow students to register for events. Send confirmation emails and manage waitlists.
- Attendance Tracking: Enable event organizers to track attendance via QR code scanning or manual input.
- Certificate Generation: Automatically generate and issue digital certificates to attendees after the event, powered by a template tool.
- Admin Dashboard: Provide administrators with tools to create, edit, and manage events, users, and certificates.
- Personalized Event Feed: Each student can create a profile expressing their event preferences; an LLM acts as a tool, and uses this profile to rank and prioritize events in the student's feed. Events closely aligned with stated preferences are placed higher in the list.

## Style Guidelines:

- Primary color: Deep violet (#673AB7) to represent knowledge and sophistication.
- Background color: Light gray (#ECEFF1) for a clean, modern interface.
- Accent color: Soft orange (#FFAB40) to highlight calls to action and important information.
- Body and headline font: 'PT Sans', a humanist sans-serif combining modern and warmth, suitable for headlines or body text
- Use clear and consistent icons from a library like Material Icons for navigation and event categories.
- Implement a responsive grid layout to ensure the application works seamlessly on different devices.
- Use subtle transitions and animations to provide feedback to user interactions and enhance the user experience.
